# runtime.coding-project
## Overview :
The goal to build this app is to learn basic functionality or concepts of React Native. and it is created using ExpoCLI.

## Installation :
* Clone repository and install dependencies:
``` 
$ git clone https://github.com/kb-codes/runtime.coding-project.git
$ cd runtime.coding-project
$ npm i
```

* Run application :
```
$ npm start
```
